Tentang

Study Club

Study Club adalah kegiatan belajar dan sharing ilmu mahasiswa Fakultas Ilmu Komputer (FASILKOM) di bidang Programming, Networking dan Multimedia. Untuk meningkatkan akademik mahasiswa dan mempersiapkan diri untuk mengikuti perlombaan.

Bidang

Programming

Mobile Programming

IonicFramework: Angular

Hal hebat tentang Ionic adalah bahwa dengan satu basis kode, Anda dapat membangun untuk platform apa pun hanya menggunakan HTML, CSS, dan JavaScript. Ikuti terus saat kita mempelajari dasar-dasar pengembangan aplikasi Ionic dengan membuat langkah aplikasi realistis secara bertahap.

Web Programming

Html, Css, JavaScript, MYSQL, PHP

Web programming terdiri dari dua suku kata yaitu web dan programming. Programming dapat diartikan sebagai proses, cara, atau pembuatan suatu program. Sedangkan web dapat diartikan sebagai jaringan komputer yang terdiri dari kumpulan situs internet yang menawarkan teks, grafik, suara dan sumber daya animasi melalui protokol transfer hypertext (HTTP). Jadi, web programming adalah suatu proses, cara, atau pembuatan suatu program berbasis web.

C++ Programming

Sharing belajar yang berhubungan dengan programming seperti algoritma dan dasar pemrograman yang menggunakan bahasa pemrograman C++.

Multimedia

Videography

Media untuk merekam suatu kejadian yang dirangkum dalam sebuah sajian gambar dan suara yang dapat kita nikmati dikemudian hari, baik sebagai sebuah kenangan ataupun sebagai bahan kajian untuk mempelajari apa yang sudah atau pernah terjadi.

Photography

Proses melukis/menulis dengan menggunakan media cahaya.
Sebagai istilah umum, fotografi berarti proses atau metode untuk menghasilkan gambar atau foto dari suatu objek dengan merekam pantulan cahaya yang mengenai objek tersebut pada media yang peka cahaya.

Networking

Networking

Networking atau Infrastruktur jaringan adalah sekumpulan komponen fisikal dan logikal yang menyediakan dasar untuk konektivitas, keamanan, routing, pengaturan, akses, dan fitur integral pada jaringan.
